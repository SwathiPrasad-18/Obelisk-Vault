Obelisk Vault is a secure, blockchain-based eVault system designed to store, verify, and manage sensitive legal or personal documents. It integrates Zero-Knowledge Proofs (ZKP), encryption, and access-controlled workflows to guarantee confidentiality, authenticity, and tamper-proof document management.

Core Features
1. Secure Document Storage

Encrypted file storage using AES-256.

Immutable metadata stored on blockchain.

Document integrity protected with Merkle trees.

2. Zero-Knowledge Verification

ZKP-based proof generation for document authenticity.

Enables verification without revealing the document contents.

3. Role-Based Access Control

User roles: Owner, Verifier, Auditor.

Multi-factor authentication options (password, biometrics).

Optional OAuth integration (future).

4. Blockchain Integration

Smart contracts for:

Document registration.

Access requests.

Proof validation.

Audit tracking.

Supports modular L1/L2 backend (Ethereum, Polygon, or private chain).

5. Interactive Web Interface

React/HTML frontend with animated blockchain-style UI.

Drag-and-drop document upload.

Document activity logs and proof display.
